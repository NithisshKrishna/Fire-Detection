DETECTION OF FIRE AND PREDICTION IF IT CAN BE EXTINGUISHED

  • Predict If the fire can be extinguished or not
  • Fire Detection Using Vedio

This Web application enables users to detect fire in a vedio by splitting the vedio into frames and also provides another link to predict if it can be extinguished or not.

The links on the navbar above: Using Vedio would require a Vedio to be uploaded to detect Fire.
Extinguishion Prediction would require SIZE,FUEL,DECIBEL,AIRFLOW,FREQUENCY and DISTANCE values
